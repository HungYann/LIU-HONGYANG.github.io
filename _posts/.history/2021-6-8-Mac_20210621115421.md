Mac 快速打开 finder:

- command + shift +h

- command + option + space

打开表情 ctrl + command + space

窗口方法缩小： ctrl + command + F

Mac 快速打开~/路径: command + shift + h

设置在 finder 标题栏显示完整路径:

```
defaults write com.apple.finder _FXShowPosixPathInTitle -bool YES
```
