systemd配置文件存在的路径：

```shell
cd /usr/lib/systemd/system/
```



systemd文件配置：

基本的配置文件由三部分组成，

Unit, Service和Install









停止：

```
 systemctl stop connectors-agglog-yundun.service		
```



启动：

```
systemctl start connectors-agglog-yundun.service	
```
