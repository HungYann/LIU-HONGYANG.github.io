

### 下载Jekyll

```
ruby -v
gem -v
gem install jekyll bundler
```

下载相关的附件：

```
 bundle install --path vendor/bundle
```





运行程序：

```
bundle exec jekyll serve
```





创建一个_drafts 文件

```
bundle exec jekyll serve --draft
```





permalink:设置一个新的url



```
permalink: "/my-new-url/test/test2"
```
